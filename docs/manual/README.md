# AutomaticWriter マニュアル作成フォルダ

このフォルダは、AutomaticWriter の利用マニュアルを作成・管理するための専用フォルダです。

## ファイル構成

- `01-user-manual-draft.md`
  - 利用者向けマニュアルの本文ドラフトです。
  - 画面操作の説明、初期設定、記事生成、WordPress投稿、自動投稿、トラブル対応をここにまとめます。
- `transcript-notes.md`
  - ミーティング音声を文字起こしした内容、重要な発言、マニュアルへ反映する要点を整理するメモです。
- `audio/README.md`
  - 音声ファイルの置き場所と取り扱いルールです。

## 音声の活用手順

1. ミーティング音声を `docs/manual/audio/` に置く、または保存場所を `transcript-notes.md` に記録します。
2. 音声を文字起こしします。
3. 文字起こしから「利用者が実際に迷いやすい箇所」「説明順」「画面名」「注意点」を抜き出します。
4. `01-user-manual-draft.md` に反映します。

## 作成方針

- 初めて使う人が、上から順に読めば設定から投稿まで進められる構成にします。
- 開発者向けの実装メモではなく、利用者向けの操作マニュアルとして書きます。
- APIキー、パスワード、トークンなどの秘密情報は記載しません。
- 画面名やボタン名は、実際のアプリ表示に合わせて更新します。
